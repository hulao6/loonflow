# 项目路线图

## 3.2.0（2026年4-5月）

**MCP Server 能力上线**

对外提供工单标准化接口，支持通过 MCP 协议完成工单创建、处理、状态查询等核心操作，为外部系统集成与 AI Agent 调用奠定基础。

## 中期规划（2026年底）

### 1. 体验与基础能力深化

- **全文检索升级**：支持多条件组合检索、搜索结果高亮与快速筛选
- **工作流引擎增强**：支持工作流分类管理、可视化编排
- **权限体系精细化**：实现字段级、操作级权限控制
- **工单关联能力**：支持自动生成子工单、工单间关联关系管理
- **表单能力扩展**：新增更多字段类型（如级联选择，各种套件）

### 2. 智能数据分析

- **工单趋势分析**：支持多维度聚合统计与可视化报表
- **处理效率洞察**：识别瓶颈环节，提供处理时长分布与 SLA 达成率分析
- **自定义看板**：支持业务方按需配置数据监控面板

### 3. AI 深度集成

- **智能填单与路由**：基于历史工单与用户描述，自动推荐工单类型、优先级及处理人
- **辅助回复建议**：结合知识库为客服提供标准应答与解决方案推荐
- **工单摘要生成**：自动生成处理过程摘要与结单总结，降低人工撰写成本
